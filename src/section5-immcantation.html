What is immcantation?

A standardized framework for analyzing B cell and T cell repertoire data.

How is immcantation used?

  • Manually (Python/R)
  • Via Nextflow pipeline (airrflow)

Manually

  • Alakazam (clonal lineages)
  • Change-O (Ig cluster assignment)
  • Dowser (lineage trees and visualization)
  • pRESTO (raw sequence processing)
  • SCOPer (clonal relationships)
  • SHazaM (SHM analysis)
  • TIgGER (novel allele identification)

Caveats

  • Tutorials mainly use docker container
  • Package management is difficult
  • Hard to Parallelize

Pipeline Caveats

  • Less control over each step
  • LSF difficulties
  • Not Space efficient
